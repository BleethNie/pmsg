# 发消息命令行程序

pmsg是一个发送消息的小工具。

#### 支持的消息类型

* 模板消息 - 微信公众号
* 一次性订阅消息 - 微信公众号
* 订阅通知 - 微信公众号
* 客服消息 - 微信公众号、微信小程序
* 订阅消息 - 微信小程序
