### 下载

`pmsg` 可运行于 Linux, macOS, FreeBSD, Windows 操作系统。

#### 使用二进制发行版

1. 下载 `pmsg` [最新版本](https://github.com/lenye/pmsg/releases)

1. 开始运行它:

   windows

    ```shell
    pmsg.exe --help
    ```

   linux

    ```shell
    ./pmsg --help
    ```

#### 源代码

```shell
git clone https://github.com/lenye/pmsg.git
```
